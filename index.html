Proxy CORS — Necesario para GitHub Pages REQUERIDO
GitHub Pages bloquea las llamadas directas a APIs externas (CORS).
Despliega un Cloudflare Worker gratuito en 2 minutos para solucionar esto:
Pasos para desplegar el proxy (gratis)
1. Ve a workers.cloudflare.com → crear cuenta gratis
2. Clic en Create Worker
3. Borra todo el código y pega el código de abajo
4. Clic en Deploy
5. Copia la URL que te da (ej: https://piporadio-proxy.TU-USUARIO.workers.dev)
6. Pégala aquí abajo
📋 Código del Worker — copia y pega en Cloudflare
export default { async fetch(request) { const url = new URL(request.url); const target = url.searchParams.get('target') || url.pathname.replace('/proxy/', ''); const apiUrl = 'https://udioapi.pro' + url.pathname + url.search; const headers = new Headers(request.headers); headers.set('Access-Control-Allow-Origin', '*'); if (request.method === 'OPTIONS') { return new Response(null, { headers: { 'Access-Control-Allow-Origin': '*', 'Access-Control-Allow-Methods': 'GET,POST,OPTIONS', 'Access-Control-Allow-Headers': '*', }}); } const res = await fetch(apiUrl, { method: request.method, headers: { 'Authorization': request.headers.get('Authorization') || '', 'Content-Type': 'application/json', }, body: request.method !== 'GET' ? request.body : undefined, }); const data = await res.text(); return new Response(data, { status: res.status, headers: { 'Content-Type': 'application/json', 'Access-Control-Allow-Origin': '*', } }); } };
